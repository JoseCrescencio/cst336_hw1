Where is this OS used?

The MacOS operating system is exclsively found on macintosh computers. Although, it can be emulated on other devices, the process requires some technical experience. This means that this operating system has a smaller customer base than Windows OS.

A bit of History

  1. OSX 10.0 - CHEETAH (2000)

    Apple's new GUI that introduced applications like textedit, preview and quicktime.

  2. OSX 10.2 - JAGUAR (2002(

    Introduced universal accesss for those with physical impairments and contactbook. First time apple logo was used on startup.

  3. OSX 10.3 - PANTHER (2003)

    Introduced Safari and increased compatibility with Windows devices.

  4. OSX 10.4 - TIGER (2005)

    Introduced spotlight search and was the OS installed the first macintosh computers with an intel processer.

  5. OSX 10.6 - SNOW LEOPARD (2009)

    Introduced tha appstore and dropped support for PowerPC architecture. Built on the support for 64-bit systems, Time machine backup, and Boot Camp provided by OSX 10.5.

  6. OSX 10.8 - MOUNTAIN LION (2012)

    Ported messages app, notification center, and reminders from ios. Built on icloud, which was introduced in OSX 10.7.

  7. OSX 10.10 - YOSEMITE (2014)

    Brought a big design change to the desktop. It also introduced a handoff feature to exchange tasks on to other devices.

  8. OSX 10.13 - HIGH SIERRA (2017)

    Improved Apple file system security and efficiency. It also improved fidelity of images and safari web browser.

Sources: An Illustrated History of macOS
